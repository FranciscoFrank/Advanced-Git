Як переглянути список тегів?
git tag -l / git tag

Як додати зміни до вже існуючого коміту (одною командою)?
git commit -a --amend --no-edit

Як скасувати останній коміт?
git reset --soft HEAD~1

Що таке теги в Git та для чого вони використовуються?
Теги — це специфічні позначки на важливих моментах в історії комітів. Використовуються, щоб створити певну точку в історії комітів, зазвичай для маркування версії релізу.

Як додати всі файли до гітігнору з папки /web, окрім файлу /web/site?
/web/*
!/web/site

Як створити тег та додаткові параметри?
git tag <>
Додаткові параметри:

-f — перезаписує поточний тег
-d — видаляє тег
-a — створює анотацію до тега (опис)
Як видалити файл з git, але залишити його у файловій системі?
git rm --cached <ім'я_файлу>

Як змінити останній коміт?
git commit --amend

Як перейти до конкретного тега?
git checkout <ім'я тегу>

Як скасувати мердж з конфліктами?
git merge --abort
