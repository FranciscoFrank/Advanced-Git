Як переглянути список тегів?
git tag -l

Як додати зміни в уже існуючий коміт (одною командою)?
git commit -a --amend --no-edit

Як відмінити останній коміт?
git reset --soft HEAD~1

Що таке теги в Git та для чого вони використовуються?
Теги - це специфічні позначки для вказання певниз версій в історії комітів. Використовуються щоб створити певну точку в історії комітів, зазвичай для маркування версії релізу

Як додати всі файли до гітігнору з папки /web окрім файлу /web/site?
/web/*
!/web/site

Як створити тег та додаткові параметри?
git tag <>
Додаткові параметри:
1. -f - перезаписує поточний тег
2. -d - видаляє тег
3. -a - створює анотацію до тега (опис)

Як видалити файл з git але залишити його в файловій системі?
git rm --cached <ім'я_файлу>
/
Як змінити останній коміт?
git commit --amend

Як перейти до конкретного тега?
git checkout <ім'я тегу>
