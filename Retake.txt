Як переглянути список тегів?
git tag -n

Як додати зміни в уже існуючий коміт (одною командою)?
git commit -a --amend --no-edit

Як відмінити останній коміт?
git reset --soft HEAD~1

Як додати всі файли до гітігнору з папки /web окрім файлу /web/site?
/web/*
!/web/site

Як створити тег та додаткові вказівники?
git tag -a <ім'я тегу> -m "Опис тегу"
